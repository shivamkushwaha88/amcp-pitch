01 — The Vision

What is this, really?

We're building a platform where people can find the exact kind of human connection they need — right now, today — without any identity risk. Not a dating app. Not random chat. Something genuinely new.

"Meet someone who matches what you need today."

Loneliness is a documented global crisis. People want real conversation, study partners, accountability buddies, or just someone to vent to — but existing platforms either require identity exposure or devolve into inappropriate spaces. We're filling that gap.

The platform philosophy rests on four pillars:

🔒

Private

No name, no photo, no identity reveal — ever. Total anonymity by design.

🛡️

Safe

AI moderation, reputation scoring, and fast reporting keeps bad actors out.

👤

Anonymous

When the chat ends, it's gone. No trails, no records, no awkwardness.

💬

Meaningful

Intent-first matching ensures conversations have purpose and direction.

03 — Safety & Moderation

Safety is the product.

Without trust, this platform dies. So safety isn't a feature — it's the foundation. We build three layers of protection:

🤖

AI Chat Moderation

Real-time detection of sexual solicitation, harassment, spam, and privacy invasion. Powered by Claude API. Automatic warnings, restrictions, and bans.

⭐

Hidden Reputation Score

Every user has a trust score. Good behavior raises it. Reports lower it. Good users get matched with good users. Bad actors get isolated then removed.

🚩

One-Tap Reporting

Every chat has a visible report button. Fast action on repeat abuse. Zero tolerance for harassment or identity solicitation.

07 — Build Timeline

MVP in 8–11 weeks.

The platform is broken into clean phases. Each phase delivers a working, testable product. No phase is wasted work.

Week 1–3 · Phase 1

Core UI + Intent Selection + Basic Chat

Full frontend, intent selection screen, basic anonymous chat flow, Socket.IO real-time messaging, matching logic.

Week 4–6 · Phase 2

AI Moderation + Reputation System + Reporting

Claude API integration for chat monitoring, reputation score engine, user reporting system, ban/warning system.

Week 7–8 · Phase 3

Payments + Premium Features

Razorpay integration, premium subscription flow, priority matching queue for premium users, premium badges.

Week 9–11 · Phase 4

Testing + Polish + Soft Launch

Bug fixes, performance optimization, mobile responsiveness, stress testing, beta user feedback, public launch.
